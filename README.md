# 闪电号（仿今日头条后台管理系统）

## 项目克隆

```shell
$ git clone https://github.com/load-more/headline-admin.git
```

## 项目初始化

```shell
$ npm install
```

## 运行项目

```shell
$ npm run serve
```

## 打包项目

```shell
$ npm run build
```
